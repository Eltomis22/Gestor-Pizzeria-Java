# Gestor de Pizzeria en java

Este proyecto es una aplicacion de consola en Java diseñada para la gestion de pedidos de una pizzeria. El principal objetivo es demostrar y comparar la implementacion de 3 algoritmos distintos: Insercion, Shellsort y Quicksort, aplicados a diferentes atributos de la clase Pedido.

## Funcionalidades

- Tiempo de preparacion
- Precio total
- Nombre del cliente

## Cómo usar

1. Compilar los archivos Java
2. Ejecutar la clase Main
3. Seguir las instrucciones en pantalla para gestionar las tareas
